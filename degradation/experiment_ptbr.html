Um Experimento sobre Degradação Estrutural em Sistemas Complexos


Nesta página apresentamos os passos necessários para a execução do experimento sobre degradação estrutural em sistemas complexos.

Preparação do Ambiente

  • 1º Passo: Selecione o projeto de software que será utilizado neste experimento.
  • 2º Passo: Caso você não tenha o código fonte em seu computador, faça o download do código fonte em seu computador. Solicitamos que esse procedimento seja realizado por meio do git, pois utilizaremos dados do repositório disponibilizados pelo git.
  • 3º Passo: Faça o download do software de análise de código que será utilizado neste experimento. O software está disponível no seguinte link [donwload].
  • 4º Passo: Realize a extração do software em um diretório com permissão de escrita e leitura.
  • 5º Passo: Para que o sistema funcione adequadamente, é necessário que a máquina virtual Java esteja instalada e configurada. O software foi desenvolvido na versão 8 (1.8) do Java.
  • 6º Passo: Faça o download do material suplementar disponibilizado para apoiar a execução do experimento. O material está disponível no seguinte link [donwload]

Execução do Experimento

Após a configuração do ambiente, precisamos realizar um treinamento sobre os conceitos envolvidos no experimento e sobre a execução do experimento. Nesse sentido, solicitamos que assista aos seguintes vídeos:
  • 1º Apresentação dos Conceitos Básicos [vídeo]
  • 2º Explicação sobre a Execução do Experimento [vídeo]

Execução do Experimento

Estimamos que este experimento deve demorar de 30 minutos a 1 hora. Porém, você pode levar o tempo que achar necessário. Assim que estiver pronto para iniciar o experimento, por gentileza, seguir os seguintes passos:
  1. Preencher o formulário de caracterização: [formulário]
  2. Executar o software do experimento, conforme as instruções do treinamento.
  3. O software vai gerar 6 casos para análise, que serão exibidos individualmente.
  4. Para cada caso, realizar a análise e preencher o seguinte formulário: [formulário]
  5. É importante notar que não será possível voltar em um caso após finalizar. Sendo assim, o formulário do experimento deve ser preenchido imediatamente para cada caso analisado.
  6. Após a análise dos 6 casos, preencher o formulário pós-experimento: [formulário]
