A Study on Structural Degradation in Complex Systems


On this page we present the necessary steps for the execution of the study on structural degradation in complex systems.

Environment Setup

  • 1st Step: Select the software project to be used in this study.
  • 2nd Step: If you don't have the source code of the project on your computer, download the source code on your computer. We request that this procedure be performed through git, as we will use data from the repository made available by git.
  • 3rd Step: Download the static analysis software that will be used in this study. The software is available at the following link [donwload]
  • 4th Step: Extract the software in a directory with write and read permission.
  • 5th Step: For the software to function properly, it is necessary that the Java virtual machine is installed and configured. The software was developed in Java version 8 (1.8).
  • 6th Step: Download the supplementary material made available to support the execution of the study. The material is available at the following link [donwload]

Trainning

After setting up the environment, we need to conduct a training on the concepts involved in the study and on the execution of the study. In this regard, we ask that you watch the following videos:
  • 1st Basic Concepts [video]
  • 2nd Explanation of the Study Execution [video]

Study Execution

We estimate that this study should take from 30 minutes to 1 hour. However, you can take the time you think is necessary. Once you are ready to start the study, please follow the steps below:
  1. Fill out the characterization form: [form]
  2. Run the static analysis software, according to the training instructions.
  3. The software will generate 6 cases for analysis, which will be displayed individually.
  4. For each case, perform the analysis and fill out the following form: [form]
  5. It is important to note that it will not be possible to return to a case after finalizing. Therefore, the study form must be completed immediately for each analyzed case.
  6. After analyzing the 6 cases, fill out the post-experiment form: [form]
