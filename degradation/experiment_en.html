A Study on Structural Degradation in Complex Systems


On this page we present the necessary steps for the execution of the study on structural degradation in complex systems.

Environment Setup

  • 1st Step: Select the software project that you will use this study.
  • 2nd Step: If you don't have the source code of the project on your computer, download it on your computer. You should use git to download the source code since we will need metadata from the repository.
  • 3rd Step: Download the static analysis software that yout will use in this study. The software is available at the following link [download]
  • 4th Step: Extract the software in a directory that has write and read permission.
  • 5th Step: For the software to function properly, you should have the Java virtual machine installed and configured on your machine. The software was developed in Java version 8 (1.8).
  • 6th Step: Download the supplementary material to support the execution of the study. The material is available at the following link [download]

Training

After setting up the environment, we need to conduct a training session. This session aims to explain the study's concepts and its execution. Thus, we ask you to watch the following videos as part of the training session:
  • 1st Basic Concepts [video]
  • 2nd Explanation of the Study Execution [video]

Study Execution

We estimate that this study should last between 30 and 60 minutes. However, you can take your time - use as much time as you think is necessary. Once you are ready to start the study, please follow the steps below:
  1. Fill out the characterization form: [form]
  2. Run the static analysis software, according to the training instructions.
  3. The software will generate 6 cases for analysis, which will be displayed individually.
  4. For each case, perform the analysis and fill out the following form: [form]
  5. It is important to note that you will not be able to return to a case after finalizing. Therefore, the study form must be completed immediately for each analyzed case.
  6. After analyzing the 6 cases, fill out the post-experiment form: [form]
