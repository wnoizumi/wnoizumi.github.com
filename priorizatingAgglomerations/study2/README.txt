--------------------------------------------------------------
Ranking Architecturally Critical Agglomerations of Code Smells
--------------------------------------------------------------


This folder contains the supplementary material for the experiment with novice software engineering students (Study 2).


Files:

1_instructions-and-form_analysis-of-agglomerations.pdf: A pdf summarizing (1) the instructions we used for configuring the enviroment for each participant, and (2) the form we used for collecting the experiment results.

2_SportsTracker-5.7.0-src.zip: The source code version of the SportsTracker application, which was analyzed by the participants.

3_raw-results.xlsx: A spreadsheet summaring the answers provided by the participants.
